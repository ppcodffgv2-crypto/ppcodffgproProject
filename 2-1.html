ศึกษาต่อระดับปริญญาตรี (สายปวช.)

มหาวิทยาลัยที่เปิดโอกาสให้เด็กสายอาชีพ

  • มหาวิทยาลัยเทคโนโลยีราชมงคล (RMUT) ทุกแห่ง
  • มหาวิทยาลัยเทคโนโลยีพระจอมเกล้า (ลาดกระบัง, ธนบุรี, พระนครเหนือ)
  • สถาบันเทคโนโลยีไทย-ญี่ปุ่น (TNI)
  • มหาวิทยาลัยรัตนบัณฑิต (RBAC)
  • มหาวิทยาลัยรังสิต
  • มหาวิทยาลัยธุรกิจบัณฑิตย์ (DPU)
  • มหาวิทยาลัยสยาม

สาขาที่เรียนได้

  • วิทยาการคอมพิวเตอร์ (Computer Science)
  • เทคโนโลยีสารสนเทศ (IT)
  • วิศวกรรมคอมพิวเตอร์
  • วิศวกรรมซอฟต์แวร์
  • ระบบสารสนเทศเพื่อองค์กร
  • การจัดการเทคโนโลยีสารสนเทศ

ช่องทางการสมัคร

  • โควตาสายอาชีพ
  • รอบรับตรง
  • ระบบ TCAS (รอบที่ 1 และ 2 เหมาะกับเด็กสายอาชีพ)

ทางที่ 1: โควตาสายอาชีพ / รอบรับตรง

สิ่งที่ต้องเตรียม:

  • ใบจบ ปวช. หรือหนังสือรับรอง
  • ผลการเรียน (ปพ.1) หรือใบ รบ.
  • แฟ้มสะสมผลงาน (Portfolio)
  • สอบสัมภาษณ์ (บางแห่งไม่มีข้อเขียน)
  • เอกสารประกอบ เช่น สำเนาบัตร, รูปถ่าย

มหาวิทยาลัยที่นิยมใช้:

  • มหาวิทยาลัยเทคโนโลยีราชมงคล
  • มหาวิทยาลัยพระจอมเกล้า
  • ม.รังสิต, ม.สยาม, DPU

การสอบ:

  • บางแห่งใช้แค่สัมภาษณ์ + Portfolio + GPA
  • บางแห่งสอบพื้นฐานคอมพิวเตอร์/คณิต

ทางที่ 2: ระบบ TCAS

รอบ TCAS:

  • รอบ 1 Portfolio – ใช้แฟ้มสะสมผลงาน
  • รอบ 2 โควตา – ใช้ GPA และเกณฑ์เฉพาะ
  • รอบ 3–5 สำหรับสายสามัญ (บางแห่งเปิดให้สายอาชีพ)

สิ่งที่ต้องมี:

  • GPA ปวช. (ขั้นต่ำแต่ละคณะต่างกัน)
  • Portfolio (ถ้ามี)
  • บางคณะอาจสอบวิชาเฉพาะ (เช่น คณิต, วิทย์)

ถ้าต้องสอบเอง:

  • สอบคณิตพื้นฐาน
  • ความรู้ทั่วไปด้านคอมพิวเตอร์
  • ภาษาอังกฤษพื้นฐาน

เอกสารสมัครที่ควรเตรียม

  • สำเนาวุฒิการศึกษา (ปวช. หรือใบรับรอง)
  • สำเนาทะเบียนบ้าน / บัตรประชาชน
  • รูปถ่าย
  • ผลสอบ เช่น TOEIC (ถ้ามี)
  • Portfolio (ถ้าสมัครรอบ 1)

เคล็ดลับสำหรับสาย ปวช. ที่อยากเข้าป.ตรีสายคอมฯ

  • เลือกมหาวิทยาลัยที่เน้นรับสายอาชีพ
  • เตรียม Portfolio ให้ดี (โครงงาน/ผลงานจริง)
  • เรียนรู้พื้นฐาน Python หรือ HTML เพื่อเพิ่มโอกาส
  • สมัครรอบต้น ๆ โอกาสจะสูงกว่า
